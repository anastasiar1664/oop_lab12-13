## Описание проекта

Консольное приложение для управления библиотекой, демонстрирующее основные принципы объектно-ориентированного программирования.

---

## Реализованная функциональность

### Основные возможности:
- Добавление книг в библиотеку
- Выдача и возврат книг
- Поиск книги по автору
- Просмотр доступных книг
- Статистика библиотеки

### ООП-компоненты:

1. **Book (Модель)**
   - Свойства: Id, Title, Author, Year, IsAvailable
   - Методы: CheckOut(), Return(), ToString()

2. **Library (Модель)**
   - Управление коллекцией книг
   - Методы поиска и фильтрации
   - Инкапсуляция данных

3. **LibraryService (Сервис)**
   - Бизнес-логика работы с библиотекой
   - Обработка операций выдачи/возврата
   - Генерация статистики
